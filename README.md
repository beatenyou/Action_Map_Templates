# Action_Map_Templates
![alt tag](https://github.com/beatenyou/Action_Map_Templates/blob/main/github-action-mapv1.png)

Templates to help red teams create Action Maps and MITRE ATT&CK narratives for hot washes and assessment reports. Download the templates and import them into [https://draw.io](https://draw.io/). While I have mixed feelings about Visio, these templates make it easier to create clear visuals for presenting red team actions.
